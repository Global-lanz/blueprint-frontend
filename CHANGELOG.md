# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [0.1.1] - 2025-12-29

### Added
- Initial project setup
- Angular 16 frontend application
- Authentication system with guards and interceptors
- Project management functionality
- Template management system
- Task management with card view
- Stage management with detail modals
- User management for admins
- Toast notification system
- Confirmation dialog component
- Breadcrumb navigation
- NgRx store for state management
- Bootstrap styling
- Docker support with nginx configuration

### Features
- User authentication and authorization
- Create, edit, and manage projects
- Template-based project creation
- Task tracking and management
- Stage workflow management
- Admin settings and user management
- Responsive UI with Bootstrap
- Proxy configuration for API calls
